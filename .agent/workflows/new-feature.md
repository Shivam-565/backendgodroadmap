# New Animation Feature

Step 1: Ask the user: "What are you building? Describe the section, what animates, and which library (Framer Motion / GSAP / R3F)."

Step 2: Run /gsd-discuss-phase with the user's answer to lock in the animation decisions.

Step 3: Run /gsd-plan-phase to research and produce a build plan. Show the plan and wait for approval before proceeding.

Step 4: Once the user approves, run /gsd-execute-phase to build it.
